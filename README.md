# Book Finder App

A React application that allows users to search for books by title using the Open Library API.  
The app displays book covers, titles, authors, and publication years in a responsive grid with smooth animations.

---

## Features

- Search books by title  
- Responsive 4-column grid layout  
- Displays book cover, title, author(s), and publish year  
- Handles empty search, no results, and network errors gracefully  
- Clean and modern UI with animations

---

## Technologies Used

- React (Create React App)  
- CSS3 with animations and transitions  
- Open Library Search API

---

## Getting Started

### Prerequisites

- Node.js and npm installed

### Installation

1. Clone the repository:  
   ```
   git clone https://github.com/Lohakaremihir14/book-finder-app.git
   ```
2. Navigate to the project directory:  
   ```
   cd book-finder-app
   ```
3. Install dependencies:  
   ```
   npm install
   ```
4. Start the development server:  
   ```
   npm start
   ```
5. Open `http://localhost:3000` in your browser to use the app.

---

## Usage

- Enter a book title in the search box  
- Click "Search" to fetch results  
- Browse books displayed in a grid with cover images and info below
